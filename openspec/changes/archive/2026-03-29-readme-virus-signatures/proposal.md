## 為何需要此變更

專案目前缺少公開說明文件（README.md），無法在 GitHub 上清楚傳達引擎用途、架構與使用方式。此外，特徵資料庫的 hash 格式與公開病毒樣本的對應關係也需要有集中的參考依據，方便貢獻者與使用者了解資料結構。

## 變更內容

- 新增 `README.md` 於儲存庫根目錄
- 包含專案概述、架構說明、快速開始指引、特徵 YAML 格式範例
- 收錄以公開來源為根據的病毒特徵 SHA256 hash 參考表（EICAR、WannaCry、NotPetya、Mirai）
- 附上 YouTube 教學影片連結作為學習參考資源
